01-Emmet

Plugin for faster HTML and CSS coding.

01-Child >

Creates child elements (nesting)

Example:

ul>li>a

Result:

<ul> <li><a href=""></a></li> </ul>

02-sibling +

create a next sibling element ay the same level.

Example:

div+p

Result:

<div></div> <p></p>

03-multiplicación *

repeats an element or group of elemtens multiple times.

Example:

ul>li*5

Result:

<ul> <li></li> <li></li> <li></li> <li></li> <li></li> </ul>

04-class .

Creates a class attribute.

Example:

h2.title

Result:

<h2 class="title">hola</h2> <p class="title">hello</p>

05-Classes ...

Creates multiple class attributes.

Example:

top

div.clase1.clase2.clasen

Result:

<div class="clase1 clase2 clasen"></div>

06-Id #

Creates an ID attribute.

Example:

nav#menu

Result:

<nav id="menu"></nav>

07-Text {}

Inserts text content.

Example:

buttom{click me}

Result:

<button>click me</button>

08-Custom Attribute []

Creates custom attributes.

Example:

a[href=www.google.com]

Result:

<a href="www.google.com"></a>

09-Item Numbering $

Adds consecutive numbers.

Example:

ul>li.num$*3

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10-Reverse Numbering @-

Inverts numbering order.

Example:

ul>li.num$@-*3

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11-Base Numbering @n

Defines a base number for sequences.

Example:

ul>li.num$@3*4

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climb Up ^

Moves up one level.

Example:

section>article>div^p

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping ( )

Groups elements together.

Example:

(header>nav)(section>article)

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-Lorem lorem

Generates placeholder text.

Example:

lorem*5

Result:

lorem*5

15-HTML Documents !

Creates various HTML document types.

Example:

!

Result:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Document</title> </head> <body </body </html>

16-Meta Tags meta:

Generates meta tags.

Example:

meta:vp

Result:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
